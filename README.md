# zensical-updates

Generate a dated Updates (blog) section for zensical sites as plain Markdown

## Installation

```bash
uv add zensical-updates
# or
pip install zensical-updates
```

## Usage

```python
from zensical_updates import greet

greet("world")  # -> "Hello, world!"
```

The supported public API is whatever is exported in
`zensical_updates.__all__`. Full documentation (autogenerated from
docstrings) lives at
<https://IvanAnishchuk.github.io/zensical-updates/>.

A small maintenance CLI ships alongside the library for introspection:

```bash
zensical-updates info     # version + public API surface
zensical-updates --help
```

## Development

```bash
git clone https://github.com/IvanAnishchuk/zensical-updates.git
cd zensical-updates
uv sync

# Run tests
uv run pytest

# Run lints
uv run ruff check src/ tests/
uv run ruff format --check src/ tests/
uv run ty check

# Run full pre-commit suite
uv run pre-commit run --all-files

# Run supply-chain audit
uv run python scripts/audit.py

# Build / preview the docs
uv sync --group docs
uv run zensical serve            # live preview at http://127.0.0.1:8000
uv run zensical build --strict   # what CI runs
```
